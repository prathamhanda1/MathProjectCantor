The Beauty of the Cantor Set

Firstly, introducing the idea of what a Cantor Set is, it is important to understand that it brings us to quite an counter-intuitive solution. The set is formed by excluding the middle third of a total set again and again. We start with the set of [0,1] and we exclude the middle third. Follwing that, we eclude the middle third of the first third. This might sound complex in writing but it is visualised perfectly in the simulation below. Following this idea of how weird infinities really are, this dashboard first introduces us to a small simulation of the process of creating the cantor set and the math used to represent it. Then, we move on to the more interesting questions of exploring infinity using are relatively simple set. Finally, since we (along with Cantor and countless other mathematicians) have spent so much time studying this problem, we talk about how can one use it constructively.
